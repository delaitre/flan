Basics Quantifiers Special Characters Flags
. Any character except newline * 0 or more \n Newline i Ignore case
a The character a + 1 or more \r Carriage return m ^ and $ match start and end of line
ab The string ab ? 0 or 1 \t Tab s . matches newline as well
a|b a or b {2} Exactly 2 \0 Null character x Allow spaces and comments
a* 0 or more a's {2, 5} Between 2 and 5 \YYY Octal character YYY J Duplicate group names allowed
\ Escapes a special character {2,} 2 or more \xYY Hexadecimal character YY U Ungreedy quantifiers
Default is greedy. Append ? for reluctant. \x{YY} Hexadecimeal character YY (?iLmsux) Set flags within regex
\cY Control character Y
Groups Character Classes Posix Classes Assertions
(...) Capturing group [ab-d] One character of: a, b, c, d [:alnum:] Letters and digits ^ Start of string
(?P<Y>...) Capturing group named Y [^ab-d] One character except: a, b, c, d [:alpha:] Letters \A Start of string, ignores m flag
(?:...) Non-capturing group [\b] Backspace character [:ascii:] Ascii codes 0 - 127 $ End of string
(?>...) Atomic group \d One digit [:blank:] Space or tab only \Z End of string, ignores m flag
(?|...) Duplicate group numbers \D One non-digit [:cntrl:] Control characters \b Word boundary
\Y Match the Y'th captured group \s One whitespace [:digit:] Decimal digits \B Non-word boundary
(?P=Y) Match the named group Y \S One non-whitespace [:graph:] Visible characters, except space \G Start of match
(?R) Recurse into entire pattern \w One word character [:lower:] Lowercase letters (?=...) Positive lookahead
(?Y) Recurse into numbered group Y \W One non-word character [:print:] Visible characters (?!...) Negative lookahead
(?&Y) Recurse into named group Y [:punct:] Visible punctuation characters (?<=...) Positive lookbehind
\g{Y} Match the named or numbered group Y [:space:] Whitespace (?<!...) Negative lookbehind
\g<Y> Recurse into named or numbered group Y [:upper:] Uppercase letters (?()|) Conditional
(?#...) Comment [:word:] Word characters
[:xdigit:] Hexadecimal digits
